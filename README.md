# Score System Project

Aplicacion Streamlit para gestionar scoreboard con roles `admin` y `player`.

## Ejecutar local

```bash
streamlit run app.py
```

## Deploy en Streamlit Community Cloud

1. Sube este proyecto a un repositorio de GitHub.
2. En https://share.streamlit.io/ crea una nueva app.
3. Repo: selecciona tu repositorio.
4. Branch: `main`.
5. Main file path: `app.py`.
6. Deploy.

## Dependencias

Instaladas desde `requirements.txt`.
